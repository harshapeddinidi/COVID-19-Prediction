COVID-19 PREDICTOR

A Convolutional Neural Network capable of predicting whether a patient's chest X-ray image belongs to a COVID-19 positive case, Viral Pneumonia or a normal patient.

ABOUT THE PROJECT

This API and project are based on the COVID-19 Radiography dataset available at https://www.kaggle.com/tawsifurrahman/covid19-radiography-database - such recopilation has been created by a team of researchers from Qatar, Bangladesh and some collaborators from Pakistan and Malaysia. Essentially it contains several chest X-ray images for COVID-19 positive cases along with Normal and Viral Pneumonia images. In the dataset there are 219 COVID-19 positive images, 1341 normal images and 1345 viral pneumonia images that were augmented to reach 15000 images, 12000 of them used to train the model and 3000 to test it. This particular model, which is a very simple one, reached 95% of accuracy.
